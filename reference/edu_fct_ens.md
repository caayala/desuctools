# Factor de código de enseñanza en 13 niveles

Recodificación para bases de datos disponibles en el Centro de Estudios
MINEDUC, Chile Recodificación del Código de Enseñanza en 13 niveles.

## Usage

``` r
edu_fct_ens(x)
```

## Arguments

- x:

  Un vector de códigos, típicamente variable \`COD_DEPE2\`.

## Value

A factor with 13 levels, with an attached \`"label"\` attribute.
