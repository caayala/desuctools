# Divide un data.frame en columnas

Toma una tabla y la divide y pega en sucesicas columnas para imprimir en
menor espacio.

## Usage

``` r
tabla_columnas(data, ncols = 2)
```

## Arguments

- data:

  tibble

- ncols:

  integer. NÃºmero de columnas en las que se quiere dividir la data
  original. Por defecto ncols = 2.
