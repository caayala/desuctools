# Factor de niveles de enseñanza

Recodificación para bases de datos disponibles en el Centro de Estudios
MINEDUC, Chile Niveles de enseñanza

## Usage

``` r
edu_fct_cod_ense(x)
```

## Arguments

- x:

  Un vector de códigos, típicamente variable `COD_ENSE` o `COD_ENS_X`.

## Value

A factor with 44 levels, with an attached `"label"` attribute.
