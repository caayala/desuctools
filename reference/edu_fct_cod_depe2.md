# Factor de código de dependencia del establecimiento agrupado

Recodificación para bases de datos disponibles en el Centro de Estudios
MINEDUC, Chile Código de Dependencia del Establecimiento (agrupado).

## Usage

``` r
edu_fct_cod_depe2(x)
```

## Arguments

- x:

  Un vector de códigos, típicamente variable \`COD_DEPE2\`.

## Value

A factor with 5 levels, with an attached \`"label"\` attribute.
