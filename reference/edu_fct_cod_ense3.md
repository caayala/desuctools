# Factor de niveles de enseñanza agrupado

Recodificación para bases de datos disponibles en el Centro de Estudios
MINEDUC, Chile Niveles de enseñanza con educación especial reasignado.

## Usage

``` r
edu_fct_cod_ense3(x)
```

## Arguments

- x:

  Un vector de códigos, típicamente variable \`COD_ENSE3\`.

## Value

A factor with 7 levels, with an attached \`"label"\` attribute.
