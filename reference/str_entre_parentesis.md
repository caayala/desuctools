# Extrae string entre paréntesis

Función para extraer texto presente entre paréntesis. Creado para
trabajo con etiquetas de variables y extraer así el concepto preguntado.

Si no encuentra texto entre paréntesis, devuelve el texto original.

## Usage

``` r
str_entre_parentesis(text)
```

## Arguments

- text:

  string

## Value

string
