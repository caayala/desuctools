# Factor de niveles de enseñanza agrupado

Recodificación para bases de datos disponibles en el Centro de Estudios
MINEDUC, Chile Niveles de enseñanza agrupados

## Usage

``` r
edu_fct_cod_ense2(x)
```

## Arguments

- x:

  Un vector de códigos, típicamente variable \`COD_ENSE2\`.

## Value

A factor with 8 levels, with an attached \`"label"\` attribute.
