# Factor de ruralidad

Recodificación para bases de datos disponibles en el Centro de Estudios
MINEDUC, Chile Índice de ruralidad del establecimiento.

## Usage

``` r
edu_fct_rural_rbd(x)
```

## Arguments

- x:

  Un vector de códigos, típicamente variable \`RURAL_RBD\`.

## Value

A factor with 2 levels, with an attached \`"label"\` attribute.
