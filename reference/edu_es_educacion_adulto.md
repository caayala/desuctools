# Detecta códigos de enseñanza para adultos

Recodificación para bases de datos disponibles en el Centro de Estudios
MINEDUC, Chile Detecta niveles de enseñanza para adultos ya que su
segunda cifra significativa es igual al dígito \`6\`.

## Usage

``` r
edu_es_educacion_adulto(x)
```

## Arguments

- x:

  Un vector numérico o de cadenas.

## Value

Un vector lógico indicando si es un código de enseñanza para adultos.
