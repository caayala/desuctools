# Codigos de enseñanza

Descripción y equivalencias entre codigos de enseñanza utilizados por
MINEDUC de Chile para la clasificación de escuelas.

## Usage

``` r
codigos_ensenanza
```

## Format

data.frame con 44 filas (codigos de enseñanza) y 6 variables

- cod_ense:

  Codigo de enseñanza

- descripcion:

  Descripción de codigo de enseñanza

- cod_ense2:

  Codigo de enseñanza agrupados 2, distingue educación especial

- cod_ense2_lab:

  Descripción de codigo de enseñanza 2

- cod_ense3:

  Codigo de enseñanza agrupados 3

- cod_ense3_lab:

  Descripción de codigo de enseñanza 3
