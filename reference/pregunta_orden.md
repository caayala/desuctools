# Orden de preguntas según valor de categoría

función de ayuda para \`forcats::fct_reorder2\` con tal de ordenar
distintas preguntas según una catogoría de respuesta de interès
\`cat_orden\`.

## Usage

``` r
pregunta_orden(.x, .y, cat_orden)
```

## Arguments

- .x:

  chr o factor. Variable con categorías de respuesta.

- .y:

  numeric. Valor de respuesta para cada categoría de respuesta

- cat_orden:

  string. Nombre de la categoría de respuesta de interés.

## Value

numeric vector
